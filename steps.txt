Steps to create and start developeing a laravel project

1. er model
2. generate models
3. create controllers (controllers are the classes that can handles communication between models and return responses)
4. create routes for RESTful API
5. create attributes for the models (define relationships)
6. implementing migration
7. implementing migration for RESTful API in pivot table
8. create laravel factories for database seeding
9. Issue all code which need to be executed during database seeding in 'seeds\DatabaseSeeder.php' file
